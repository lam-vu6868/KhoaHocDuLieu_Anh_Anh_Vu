## 2. TẠI SAO GHÉP SAI → PHÌNH?

### ❌ Ví Dụ SAI: Direct Merge

```python
# ❌ KHÔNG LÀM CÁI NÀY:
df_merged = df_id.merge(df_data, on=['item_id', 'shop_id'])
df_final = df_merged.merge(df_rating, on='item_id')  # ☠️ PHÌNH!

# Kết quả:
item_id=1 (iPhone 15)
  → Row 1: (1, 5M, 4.8⭐, comment_A)
  → Row 2: (1, 5M, 4.8⭐, comment_B)   ← Lặp item_id=1
  → Row 3: (1, 5M, 4.8⭐, comment_C)   ← Lặp lần 3
  ...
  → Row 20: (1, 5M, 4.8⭐, comment_T) ← Tổng 20 row cho 1 sản phẩm!

Vấn đề:
- 1 sản phẩm → 20 row (dữ liệu sản phẩm lặp 20 lần)
- Thông tin sản phẩm (brand, price, rating trung bình) bị lặp → BỎ PHÍ BỘNG NHỚ
- Khi phân tích: "số sản phẩm bán" = 20 thay vì 1 → TÍNH SAI!
```

### ✅ Ví Dụ ĐÚNG: Aggregate Trước

```python
# ✅ ĐÚNG:
# BƯỚC 1: Tóm tắt RATING từ 61,472 → 3,060 rows
rating_agg = df_rating.groupby('item_id').agg({
    'rating_star': ['mean', 'std', 'count'],
    'user_name': 'count',
    'comment': lambda x: (x != '').sum()
}).reset_index()

Kết quả: 3,060 rows (1 row = 1 sản phẩm + summary rating)
item_id=1:
  - avg_rating_star = 4.8
  - std_rating_star = 0.3
  - count_ratings = 20 (tổng 20 bình luận)
  - comments_with_text = 15 (15 bình luận có text)

# BƯỚC 2: Ghép lần lượt (1:1)
df_merged = df_id.merge(df_data, on=['item_id', 'shop_id'])     # 3,060 rows
df_final = df_merged.merge(rating_agg, on='item_id')            # 3,060 rows ✅

Kết quả: 3,060 rows (1 row = 1 sản phẩm + đầy đủ thông tin)
```

## 4. PHẦN 4: GHÉP 3 SHEET (ĐỌC KỸ!)

### 🚨 LỖI PHẢI TRÁNH

```python
# ❌ SAI: Ghép trực tiếp
df_result = df_id.merge(df_data, on=['item_id', 'shop_id'])
df_result = df_result.merge(df_rating, on='item_id')
print("Rows:", len(df_result))  # ☠️ 61,200 rows (PHÌNH!)

# Vấn đề:
# - 1 sản phẩm có 20 rating → 20 rows lặp lại (brand, price, v.v.)
# - Khi phân tích: "bán 61,200 sản phẩm" → SAI (thực tế chỉ 3,060)
# - Bộ nhớ lãng phí
# - Kết quả phân tích hoàn toàn sai lệch
```

### ✅ CÁCH ĐÚNG: Aggregate → Merge

```python
# ✅ BƯỚC 1: Aggregate RATING (61,472 → 3,060)
rating_agg = df_rating.groupby('item_id').agg({
    'rating_star': ['mean', 'std', 'count'],
    'user_name': lambda x: (x != '').sum(),      # Bao nhiêu có user_name
    'comment': lambda x: (x.notna() & (x != '')).sum()  # Bao nhiêu có comment
}).reset_index()

# Rename columns để dễ hiểu
rating_agg.columns = ['item_id', 'avg_rating', 'std_rating', 'count_ratings', 
                      'count_users', 'count_comments']

print("After aggregate:", rating_agg.shape)  # (3060, 6)

# ✅ BƯỚC 2: Ghép tuần tự (đảm bảo 1:1)
df_merged = df_id.merge(df_data, on=['item_id', 'shop_id'], how='inner')
print("After merge ID+DATA:", df_merged.shape)  # (3060, 16)

df_final = df_merged.merge(rating_agg, on='item_id', how='left')
print("After merge with RATING_agg:", df_final.shape)  # (3060, 21)

# ✅ BƯỚC 3: Validate
print("\n=== VALIDATION ===")
print("Rows:", len(df_final))      # 3060 ✅
print("NULL count:\n", df_final.isna().sum())  # Kiểm tra NULL
print("Columns:", df_final.columns.tolist())
```

```

   - ID info (item_id, shop_id, shop_location, name)
   - DATA info (brand, sold_qty, prices, rating_avg, v.v.)
```

## 5. CHI TIẾT CODE KHÁM PHÁ (Full Script)

Tôi sẽ viết file `0_data_exploration.py` chứa:

```python
# 0_data_exploration.py

import pandas as pd
import numpy as np
from pathlib import Path

# Output file
output_file = 'data_exploration_report.txt'

with open(output_file, 'w', encoding='utf-8') as f:
    f.write("=" * 80 + "\n")
    f.write("DATA EXPLORATION REPORT\n")
    f.write("=" * 80 + "\n\n")
    
    # Load 3 sheets
    df_id = pd.read_excel('data.xlsx', sheet_name='ID')
    df_data = pd.read_excel('data.xlsx', sheet_name='DATA')
    df_rating = pd.read_excel('data.xlsx', sheet_name='RATING')
    
    # ===== SHEET ID =====
    f.write("\n" + "=" * 80 + "\n")
    f.write("1. SHEET ID\n")
    f.write("=" * 80 + "\n")
    f.write(f"Shape: {df_id.shape}\n")
    f.write(f"NULL count:\n{df_id.isna().sum()}\n")
    f.write(f"Duplicates: {df_id['item_id'].duplicated().sum()}\n")
    
    # ===== SHEET DATA =====
    f.write("\n" + "=" * 80 + "\n")
    f.write("2. SHEET DATA\n")
    f.write("=" * 80 + "\n")
    f.write(f"Shape: {df_data.shape}\n")
    f.write(f"NULL count:\n{df_data.isna().sum()}\n")
    f.write(f"⚠️ brand NULL: {df_data['brand'].isna().sum()} ({df_data['brand'].isna().sum()/len(df_data)*100:.1f}%)\n")
    f.write(f"⚠️ original_price=0: {(df_data['original_price']==0).sum()} ({(df_data['original_price']==0).sum()/len(df_data)*100:.1f}%)\n")
    
    # ===== SHEET RATING =====
    f.write("\n" + "=" * 80 + "\n")
    f.write("3. SHEET RATING\n")
    f.write("=" * 80 + "\n")
    f.write(f"Shape: {df_rating.shape}\n")
    f.write(f"NULL count:\n{df_rating.isna().sum()}\n")
    
    rating_per_item = df_rating.groupby('item_id').size()
    f.write(f"\n🚨 1:N MỐI QUAN HỆ:\n")
    f.write(f"  - 1 sản phẩm = trung bình {rating_per_item.mean():.1f} rating\n")
    f.write(f"  - Min: {rating_per_item.min()}, Max: {rating_per_item.max()}\n")
    f.write(f"  ⚠️ PHẢI aggregate trước merge!\n")
    
    # ===== MERGE =====
    f.write("\n" + "=" * 80 + "\n")
    f.write("4. MERGE STRATEGY\n")
    f.write("=" * 80 + "\n")
    
    # Aggregate RATING
    rating_agg = df_rating.groupby('item_id').agg({
        'rating_star': ['mean', 'std', 'count']
    }).reset_index()
    rating_agg.columns = ['item_id', 'avg_rating', 'std_rating', 'count_ratings']
    
    f.write(f"After aggregate RATING: {rating_agg.shape}\n")
    
    # Merge
    df_merged = df_id.merge(df_data, on=['item_id', 'shop_id'], how='inner')
    f.write(f"After merge ID+DATA: {df_merged.shape}\n")
    
    df_final = df_merged.merge(rating_agg, on='item_id', how='left')
    f.write(f"After merge with RATING_agg: {df_final.shape}\n")
    f.write(f"✅ NO BLOAT! (3,060 rows, not 61,200)\n")
    
    f.write("\n" + "=" * 80 + "\n")
    f.write("✅ EXPLORATION COMPLETE - READY FOR DATA CLEANING\n")
    f.write("=" * 80 + "\n")

print(f"✅ Report saved to: {output_file}")
```
